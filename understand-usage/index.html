Understanding 3rd-Party Usage

We leverage a range of third-party tools—Glide, Make, OpenAI, PDF generators, e-signature services, and more—to build and customize your app. These integrations aren't included by default, but our unified credits system makes it easy to forecast and manage any usage costs across your entire tech stack.

How Revalate Credits Work

Every integration action—Glide updates, Make operations, OpenAI calls, PDF renders, e-signatures, etc.—consumes a set number of credits. We may also leverage other third-party tools not listed here to fully customize your solution. Credits give you a simple, unified way to anticipate usage costs across your entire tech stack.

Included Credits vs. Overages

Your main plan includes a fixed package of credits each month. If you exceed that allotment, additional usage is billed at $15 per 20,000 credits.

Buying More Credits

Need extra capacity? You can purchase additional 20,000-credit bundles for $15 each—or work with us to build custom bulk-credit packages at high-volume discount rates.

User Licensing & Bulk Packages

Your base plan covers a set number of user seats. Additional users are billed at $5 per user per month, and we can create custom bulk-user packages if you need a larger team license.
